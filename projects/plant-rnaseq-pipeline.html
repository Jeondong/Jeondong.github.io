Plant-rnaseq-pipeline

Overview

Plant-rnaseq-pipeline is a reusable bulk RNA-seq analysis workflow for plant transcriptome studies, covering major steps from raw read quality control to differential expression analysis.

Purpose

It was developed to provide a practical and reusable RNA-seq workflow for plant research, integrating standard processing steps such as quality control, read trimming, alignment, read counting, and downstream differential expression analysis.

Key features

  • Reusable bulk RNA-seq workflow
  • Includes QC, trimming, alignment, read counting, and DESeq2 analysis
  • Designed for practical plant transcriptome analysis
  • Simple script-based workflow for reproducible analysis
