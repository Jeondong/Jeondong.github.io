OrganelleCleaner

Overview

OrganelleCleaner is a bioinformatics tool designed to identify and remove organelle-derived contigs from genome assemblies.

Purpose

It was developed to improve nuclear genome assemblies by filtering plastid- and mitochondria-derived contigs using graph structure and sequence-based evidence.

Key features

  • Graph- and sequence-based detection
  • Designed for genome assembly cleanup
  • Supports practical preprocessing for downstream assembly analysis
